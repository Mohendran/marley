## Project Setup and start

1. Run `mix deps.get`.
2. Create and migrate your database with `mix ecto.create && mix ecto.migrate`.
3. Run `npm install`.
4. Start Phoenix endpoint with `mix phoenix.server`.
5. Now you can visit http://localhost:4000/. That's it!.
